###Java中有四种基本数据类型
## 1. 整型 
``` 1. byte 2. short 3. int 4. long  ```
## 2.浮点型
``` 1. float 2. double ```
## 3.布尔类型
``` boolean ```
## 4.字符型
``` 
char 
占位不同
长度不同
计算机的存储单位是1024 
1024字节 = 1KB
1024kb = 1M
1024M = 1G
```

** 在java中基本数据类型都是存储在java的栈中 **
