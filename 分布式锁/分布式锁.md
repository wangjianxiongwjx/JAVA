## Redis 分布式锁的实现原理
1. 定义redis key
2. 定义一个不重复的value 例如：UUID.randomUUID().toDtring();
3. 定义一个超时间
4. 设置 key value 和 timeout 到redis 返回一个boolean
   如果返回true 则执行逻辑
   如果返回false 则直接返回
5. 释放锁
   获取redis中的value是否跟set之前的value相同，如果是则释放锁。
